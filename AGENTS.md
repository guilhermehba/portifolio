# Project Instructions
Always add "use client" at the top when using framer-motion or interactive UI.
This project is a professional developer portfolio built with Next.js, TypeScript and Tailwind CSS.

The goal is to showcase freelance projects, landing pages, systems and automation work to potential clients.

## Main stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- lucide-react for icons
- framer-motion for subtle animations

## Coding rules

- Write clean, readable and maintainable code.
- Use functional React components.
- Prefer simple solutions over complex abstractions.
- Add useful comments explaining important parts of the code.
- Do not over-comment obvious code.
- Keep components small and reusable.
- Use Tailwind classes for styling.
- Avoid inline styles.
- Avoid unnecessary dependencies.

## Folder structure

- `src/components/sections`: main landing page sections.
- `src/components/ui`: reusable visual components.
- `src/components/layout`: header, footer and layout elements.
- `src/data`: editable content such as projects, services and personal info.
- `src/lib`: utility functions.

## Visual direction

The design should feel:
- premium
- modern
- dark first
- clean
- bold
- slightly futuristic
- suitable for a freelance web developer

The site should help convert visitors into WhatsApp contacts.

## Important behavior

When generating code:
- Include comments that help future maintenance.
- Keep the code easy to edit.
- Do not create unnecessary files.
- Follow the existing structure.
- Use Portuguese text for visible website content.
- Use English for code names, variables and components.
